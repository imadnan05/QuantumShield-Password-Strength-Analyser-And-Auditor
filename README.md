# 🛡 QuantumShield Ultimate Pro

QuantumShield Ultimate Pro is an advanced **frontend-only password security analyzer** built using **HTML, CSS, and JavaScript**. It helps users evaluate password strength, detect breach exposure, generate secure passwords, and understand modern password security concepts through an interactive dashboard.

---

## 🚀 Features

### 🔐 Password Security Analysis
- Real-time password strength analysis using **zxcvbn**
- Entropy calculation
- Crack time estimation
- Risk level detection (High / Medium / Low)
- Smart improvement suggestions

### 🛡 Security Checks
- **Have I Been Pwned API** breach exposure check
- Password policy compliance checker
- Common weak pattern detection

### 🔑 Hashing Demonstration
- Real-time **MD5 Hash**
- Real-time **SHA-256 Hash**

### ⚡ Productivity Tools
- Secure password generator
- Copy password to clipboard
- Export report as text file
- Password history using localStorage
- Voice feedback using Speech Synthesis

### 📊 Dashboard UI
- Circular score meter
- Modern futuristic glassmorphism design
- Responsive layout for desktop/mobile

---

## 🧰 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- zxcvbn Password Strength Library
- CryptoJS

## 📌 How It Works

### Password Strength Engine

Uses **zxcvbn** to analyze:

* Common passwords
* Repeated patterns
* Keyboard sequences
* Predictable substitutions
* Realistic crack time

### Hashing

Displays how passwords convert into:

* MD5 hash
* SHA-256 hash

### Breach Check

Uses k-anonymity model with HaveIBeenPwned API to safely check if password appeared in known breaches.
